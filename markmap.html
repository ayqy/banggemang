工具介绍及使用方法

便捷思维导图工具可以帮助您快速将 Markdown 文本转换为可视化的思维导图。使用方法：1. 在左侧编辑器中输入 Markdown 格式的文本，使用 # 号表示不同层级的标题；2. 点击"渲染导图"按钮，右侧将自动生成对应的思维导图；3. 点击"全屏"按钮可以切换全屏模式，方便查看和展示。支持 Markdown 常用语法，包括加粗、斜体、删除线、高亮、行内代码和超链接等。思维导图支持缩放和拖拽操作，方便您查看复杂的导图结构。
