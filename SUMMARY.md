## 使用说明

### 启动项目

```bash
cd /Users/pocket/Downloads/banggemang

# 方式 1：Python
python3 -m http.server 8080

# 方式 2：Node.js
npx serve . -p 8080

# 方式 3：PHP
php -S localhost:8080
```

然后在浏览器访问：http://localhost:8080/index.html

### 访问页面

- **首页：** http://localhost:8080/index.html
- **教育工具分类：** http://localhost:8080/category/education.html
- **字数计算：** http://localhost:8080/tools/wordcount.html
- **其他工具：** 类似路径

## 已知限制

1. **去手写工具：** 需要 AI 图像处理能力，目前仅显示占位页面
2. **翻译工具：** 词典数据有限，仅支持基础词汇互译
3. **搜索功能：** 基础实现，不支持模糊搜索和拼音搜索
4. **数据持久化：** 未实现本地存储，刷新后数据丢失

## 项目亮点

1. **完整复刻：** 从布局到功能，与线上页面高度一致
2. **数据驱动：** 使用 JSON 数据管理工具和分类信息
3. **模块化：** CSS 和 JS 模块化设计，易于维护
4. **零依赖：** 纯原生实现，无需任何框架
5. **可打印：** 字帖生成工具支持打印功能
6. **语音支持：** 汉字发音工具使用浏览器语音合成 API
