# 字帖生成 (zitie_new) PRD

## 状态
调研已完成，待开发。

## 功能概述
字帖生成是一款在线工具，专为生成个性化字帖而设计。用户可以选择不同的模板、年级，快速生成适合练习书写的字帖。

## 页面结构与交互
1. **公共部分**：顶部 Header（带 Logo 和搜索框），左侧 Sidebar（工具分类导航）。
2. **主要内容区**：
   - **配置区域**：
     - 年级选择器（如下拉框或输入框，默认值 "1"）。
     - 模板选择器（如下拉框或输入框，默认值 "A"）。
     - 文本输入区域：多行文本框（Textarea），占位符为“请输入汉字，非汉字输入无效”，默认填充了一些一年级的常见汉字。
   - **操作按钮**：
     - “重置”：清空或恢复默认配置。
     - “字帖”：生成或预览字帖。
     - “打印”：调用浏览器打印功能打印字帖。
   - **介绍及说明区域**：标题“工具介绍及使用方法”，说明文案：
     1. 选择您要生成字帖的模板、年级
     2. 点击打印按钮可将字帖打印下来
