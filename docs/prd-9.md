# 便捷思维导图 (markmap) PRD

## 状态
调研已完成，待开发。

## 功能概述
便捷思维导图工具可以帮助您快速创建、编辑和分享思维导图，提升思维整理和信息管理能力。基于 Markdown 语法生成思维导图。

## 页面结构与交互
1. **公共部分**：顶部 Header，左侧 Sidebar。
2. **主要内容区**：
   - **分栏布局**：
     - 左侧或上方为 **Markdown 编辑区**（多行文本框）。包含默认的示例 Markdown 文本（包含 `#`, `##`, `-` 等层级符号）。
     - 右侧或下方为 **思维导图预览区**。
     - 按钮：“渲染导图”：点击后，读取左侧 Markdown 文本内容，并在右侧/下方渲染为可视化的思维导图树。
   - **介绍及说明区域**：标题“工具介绍及使用方法”，说明文案：
     1. 根据左侧示例格式，可以复制您的工作或学习文档，您也可当场输入文字。
     2. 一级标题前面加#、二级标题前面加##、三级标题前面加###，正文前面加-，符号和文字之间有空格。
     3. 点击渲染导图，工具会将您输入的内容生成思维导图。

## 技术实现
- 可引入开源库 `markmap-view` 等相关库，在浏览器端直接渲染思维导图。
