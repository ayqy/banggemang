## 页面布局结构
1. **左侧边栏 (Sidebar)**：
   - 宽度固定，包含工具分类列表。
   - 分类包含：全部, 图片工具, PDF转换工具, 数据换算工具, 生活娱乐工具, 教育工具 (当前选中), 文本工具, 文档转换工具, 开发工具, 视频工具, 浏览器插件。
   - 每一个分类项包含一个图标和文本。

2. **主内容区 (Main Content)**：
   - 顶部可能包含搜索框及累计使用人次。
   - 工具列表容器：使用 `display: grid`（网格布局），列宽等分为三列，每列约 `360px`，整体间距均匀。
   - **工具卡片 (Tool Card)**：
     - 尺寸：`360px` 宽，`100px` 高。
     - 样式：背景纯白 `rgb(255, 255, 255)`，边框圆角 `10px`，带轻微阴影 `box-shadow: rgba(0, 0, 0, 0.1) 0px 0px 10px 0px`。
     - 内容：左侧包含一个图标（`img`），右侧包含工具名称（如“去手写”）和简短描述。右上角可能有标签（如“NEW”, “HOT”, “推荐”）。
     - 交互：鼠标悬浮时可能光标变手型，点击整张卡片跳转至对应的 HTML 页面。
