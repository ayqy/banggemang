# 高校查询 (school) PRD

## 状态
调研已完成，待开发。

## 功能概述
高校查询应用汇聚了全国各地的知名大学，包括清华大学、北京大学等，能够帮助高考学生和家长快速找到中国境内知名大学的相关信息。

## 页面结构与交互
1. **公共部分**：顶部 Header，左侧 Sidebar。
2. **主要内容区**：
   - **分类筛选区**：
     - 单选按钮组或标签页（Tabs），包含：“全部”、“985”、“211”、“双一流A”、“双一流B”、“一流学科”。默认选中“全部”。
   - **高校数据表格**：
     - 表头包含列：“学校名称”、“985”、“211”、“双一流”。
     - 列表渲染对应标签分类下的学校及其状态（“是”或“否”）。
     - 支持分页（如果数据量很大，或全量展示）。
   - **介绍及说明区域**：标题“工具介绍及使用方法”，说明文案：
     1. 您可以选择查看985、211、一流学科等类别。
     2. 系统将为您跳转对应的高校。
