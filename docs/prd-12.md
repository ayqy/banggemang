# 翻译 (translate) PRD

## 状态
调研已完成，待开发。

## 功能概述
翻译工具可以帮助您将文本从一种语言翻译成另一种语言，支持多种外部翻译源。

## 页面结构与交互
1. **公共部分**：顶部 Header，左侧 Sidebar。
2. **主要内容区**：
   - **翻译源选择**：
     - 单选按钮或下拉框，选择翻译引擎，例如“搜狗翻译”、“有道翻译”、“百度翻译”。
   - **输入/输出区域**：
     - 若采用集成方案，则左侧为输入框，右侧为输出框。
     - 若采用 iframe 方案，直接在下方加载对应翻译引擎的页面。
     - 结合原始描述：“工具会自动为您将结果显示在相应区域”，此处可实现为一个双栏布局，模拟翻译界面（实际使用免费API或直接模拟翻译行为）。
   - **介绍及说明区域**：标题“工具介绍及使用方法”，说明文案：
     1. 您可以选择想要的翻译网站，例如搜狗翻译、有道翻译。
     2. 设置您要翻译前后的语言类型。
     3. 工具会自动为您将结果显示在相应区域。
