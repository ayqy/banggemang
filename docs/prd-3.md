# 亲戚关系计算 (relatives_name) PRD

## 状态
调研已完成，待开发。

## 功能概述
亲戚关系计算可以帮助用户快速计算两个或多个对象之间复杂的亲戚关系。操作简单易用，只需要选择具体的亲戚关系链路，即可得到准确的称呼。

## 页面结构与交互
1. **公共部分**：顶部 Header，左侧 Sidebar。
2. **主要内容区**：
   - **控制面板**：
     - 单选按钮（Radio）：选择“我是男的”或“我是女的”（默认选中男）。
     - 复选框（Checkbox）：选择是否是“对方称呼我”（默认是不勾选，即“我称呼对方”）。
   - **计算链路展示**：
     - 一个输入框或显示区域：“要找的称谓”，显示当前点击组合的关系链，例如“父亲 的 弟弟”。
   - **键盘区域**：
     - 包含按钮：“父”、“母”、“夫”、“妻”、“子”、“女”、“兄”、“弟”、“姐”、“妹”、“<”（回退/删除）。
   - **结果展示区**：
     - 底部显示最终计算得出的称呼结果。
   - **介绍及说明区域**：标题“工具介绍及使用方法”，说明文案：
     1. 首先选择是否是对方称呼我，如选择系统会生成在这种亲戚关系中对方怎样称呼您，未选择即代表您要生成在这种亲戚关系中您怎样去称呼对方。
     2. 选择您的性别。
     3. 选择您要找的称谓，比如说“弟弟的老婆”，选择之后系统会自动计算称呼并且显示在底部。
