# 词语注解 (explain) PRD

## 状态
调研已完成，待开发。

## 功能概述
词语注解工具可以帮助您查找和学习词语的详细解释和用法。通过这款工具，您可以输入任意词语，快速获取该词语的注解、拼音、词性、例句等信息。

## 页面结构与交互
1. **公共部分**：顶部 Header，左侧 Sidebar。
2. **主要内容区**：
   - **输入查询区**：
     - 单行文本输入框，左侧或上方提示文字“请输入查询汉词”。
     - 按钮：“注解查询”。
   - **结果展示区**：
     - 点击查询后，在下方区域展示词语的拼音、词性、注解解释以及可能的例句。
   - **介绍及说明区域**：标题“工具介绍及使用方法”，说明文案：
     1. 在输入框中输入您希望查询的词语。
     2. 点击“查询”按钮，工具会自动检索并显示该词语的详细注解。
     3. 在结果区域查看所查询词语的详细注解。
     4. 根据工具提供的词语注解信息，进行语言学习和理解。您可以输入更多词语，继续查询和学习。
