工具介绍及使用方法

亲戚关系计算可以帮助用户快速计算两个或多个对象之间复杂的亲戚关系。亲戚关系计算操作简单易用，只需要打开工具，选择需要计算的亲戚关系，点击“计算”按钮即可得到准确的计算结果。方便用户根据实际情况进行灵活选择和使用。
1、首先选择是否是对方称呼我，如选择系统会生成在这种亲戚关系中对方怎样称呼您，未选择即代表您要生成在这种亲戚关系中您怎样去称呼对方。
2、选择您的性别。
3、选择您要找的称谓，比如说“弟弟的老婆”，选择之后系统会自动计算称呼并且显示在底部。
