# 字数计算 (wordcount) PRD

## 状态
调研已完成，待开发。

## 功能概述
字数计算专为快速计算文本中的字数、字符数、单词数和句子数而设计。无论您是需要统计文章的字数、检查字符限制，还是进行文本分析，这款工具都能帮助您高效完成任务。

## 页面结构与交互
1. **公共部分**：顶部 Header，左侧 Sidebar。
2. **主要内容区**：
   - **输入区域**：
     - 大型多行文本框（Textarea），占位符为“请输入”。
     - 提示文本：“空格、换行不计入字数”。
   - **操作按钮**：
     - “确认”：触发计算（或支持输入时实时计算，点击确认做最终展示）。
     - “清空”：清空文本框内容及计算结果。
   - **结果展示区**：
     - 显示计算得出的：字符数、汉字数、英文单词数等信息。
   - **介绍及说明区域**：标题“工具介绍及使用方法”，说明文案：
     1. 在文本输入框中输入或粘贴您需要计算的文本内容。
     2. 工具会自动计算并显示文本的字数、字符数。
