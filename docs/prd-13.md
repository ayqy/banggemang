# 汉字偏旁 (radical) PRD

## 状态
调研已完成，待开发。

## 功能概述
汉字偏旁工具可以帮助您查找和学习汉字的偏旁部首。输入汉字后，快速获取该汉字的偏旁部首信息，了解其构造和意义。

## 页面结构与交互
1. **公共部分**：顶部 Header，左侧 Sidebar。
2. **主要内容区**：
   - **输入区**：
     - 一个多行文本框（Textarea），占位符为“输入需要查询偏旁文字”。
   - **操作按钮**：
     - “查询”按钮。
   - **输出区**：
     - 点击查询后，在下方展示输入的汉字的偏旁、部首等详细信息（可能分为多个展示框/列）。
   - **介绍及说明区域**：标题“工具介绍及使用方法”，说明文案：
     1. 在输入框中输入您希望查询的汉字。
     2. 点击“查询”按钮，工具会自动检索并显示该汉字的偏旁部首信息。
