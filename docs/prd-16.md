# 成语大全 (chengyujielong) PRD

## 状态
调研已完成，待开发。

## 功能概述
成语大全查询是一款免费的中国传统文化知识工具，它收录了大量的成语，能够根据汉字、拼音、笔画数进行快速查询。

## 页面结构与交互
1. **公共部分**：顶部 Header，左侧 Sidebar。
2. **主要内容区**：
   - **查询配置区**：
     - 单选按钮组（Radio）：查询方式选择“汉字”、“拼音”、“笔画数”。
     - 文本输入框：占位符“例如:“美”、“五十””。
     - 按钮：“查询”。
   - **结果展示区**：
     - 点击查询后，下方展示符合查询条件的成语列表或卡片。
   - **介绍及说明区域**：标题“工具介绍及使用方法”，说明文案：
     1. 您可以选择查询方式，查询方式有：汉字、拼音、笔画数。
     2. 输入查询条件：
        1）汉字查询方式：输入想要查询成语的首字
        2）拼音查询方式：输入成语首字拼音和音调，例如输入”yi4“
        3）笔画查询方式：输入想要查询成语的笔画
     3. 选择查询方式并输入查询条件结束后，点击查询，工具会为您查询相应的成语。
