# 工具箱全量测试用例 (Test Cases)

## 0. 首页 (index.html)
- **用例 1.1**：页面加载后，应该能看到 16 个工具卡片。
- **用例 1.2**：点击任一工具卡片，页面应正确跳转至对应的子 HTML 页面。
- **用例 1.3**：侧边栏“教育工具”应处于激活状态（高亮）。

## 1. 去手写 (handwriting_erasure.html)
- **用例 1.1**：页面加载后，应显示带有虚线边框的拖拽区域和说明文字。

- **用例 1.3**：包含功能说明模块，文案展示无误。
